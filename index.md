## ⚙️ How It Works

- **Data** is pulled from [Yahoo Finance](https://finance.yahoo.com/) and [GDELT](https://www.gdeltproject.org/)  
- **Pipelines** update automatically via GitHub Actions  
- **Visuals** are regenerated and published here on every run
